# pokemon_java

Proyecto sencillo de Pokemon con interfaz grafica en JavaFX.

La ventana incluye un menu superior con opciones de juego y ayuda. En `Ayuda`
puedes ver como jugar, que hace cada boton y como funcionan los tipos.

## Ejecutar

```bash
mvn javafx:run
```

## Compilar

```bash
mvn package
```
